Свойство/метод ---- Описание

is_on ---- инициализируется значение false
value ---- инициализируется значение 0
max_value ---- устанавливает максимальное значение
min_value ---- устанавливает минимальное значение
turn_on() ---- включает устройство
turn_off() ---- выключает устройство
set_value() ---- устанавливает значение устройства
get_is_on() ---- возвращает значение свойства is_on
get_value() ---- возвращает значение свойства value
